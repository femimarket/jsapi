# jsapi@4.5.1

A TypeScript SDK client for the femi.market API.

## Usage

First, install the SDK from npm.

```bash
npm install jsapi --save
```

Next, try it out.


```ts
import {
  Configuration,
  ApiHandlerApi,
} from 'jsapi';
import type { ApiHandlerRequest } from 'jsapi';

async function example() {
  console.log("🚀 Testing jsapi SDK...");
  const config = new Configuration({ 
    // Configure HTTP bearer authorization: bearer
    accessToken: "YOUR BEARER TOKEN",
  });
  const api = new ApiHandlerApi(config);

  const body = {
    // API
    aPI: ...,
  } satisfies ApiHandlerRequest;

  try {
    const data = await api.apiHandler(body);
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}

// Run the test
example().catch(console.error);
```

- Package version: `4.5.1`

- Build package: `org.openapitools.codegen.languages.TypeScriptFetchClientCodegen`

The generated npm module supports the following:
